# Contributing

Marble is currently source-available under the Elastic License 2.0. The
maintainers intend to relicense the project under the MIT License in the future.

Before submitting a contribution, read CLA.md. By opening a pull request,
sending a patch, or otherwise submitting material to Marble, you agree to the
contributor terms in CLA.md.

Please do not contribute material unless it can be licensed under both:

1. the current Elastic License 2.0 terms; and
2. the MIT License in the future.

This keeps the project usable today while preserving the planned path to a more
permissive license later.
